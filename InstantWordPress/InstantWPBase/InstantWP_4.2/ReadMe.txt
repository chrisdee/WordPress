INSTANT WORDPRESS 4.2

TO START INSTANT WORDPRESS
--------------------------

Double click on InstantWP to run Instant WordPress.

The Instant WordPress Control Panel will open. 

Please see the documentation for more information on the Instant WordPress Control Panel.


TO STOP INSTANT WORDPRESS
--------------------------

Close the Instant WordPress Control Panel to shutdown Instant WordPress.
